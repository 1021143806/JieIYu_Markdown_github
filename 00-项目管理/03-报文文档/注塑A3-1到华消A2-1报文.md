
# NOC报文（定制模块报文）

## 上架任务和回库任务

> 请求地址：http://10.68.2.19:7000/ics/taskOrder/addTask
>
> 任务报文：

```json
//1上料，2下料，3回库
{
    "shelfNumber": "B3536",
    "toWorkStation": "",//下发的点位或片区域
    "taskTypeId": 3,
    "orderId": "tes1112222222tDHB2-C213215D305DC485BB90F3673448452D3"
}
```

仅使用 1 上料和 3 回库即可

# NOC报文(如果使用标准接口)

## 上架任务和回库任务

> 请求地址：http://10.68.2.19:7000/ics/taskOrder/addTask
>
> 任务报文：

```json
{
    "modelProcessCode": "SJBY",//下架搬运XJBY；上架搬运 SJBY；货架回库 HJHK
    "priority": "6",//优先级
    "fromSystem": "NOC",
    "orderId": "13706888",
    "taskOrderDetail": [
        {
            "shelfNumber": "A20001",
            "taskPath": "12345678"  //(工作台点位)
        }
    ]
}
```

### 继续任务接口

```json
{
"orderId":"",
"shelfNumber":"B0622"//任务单号和货架编号任意选择其一即可
}
```

上架点设备负载，在上架点等待，上架结束，调用继续任务后货架回库

# WMS 报文

## 下架任务（可选在工作台放下货架释放或在工作台举升货架）

> 请求地址：http://10.68.2.19:7000/ics/taskOrder/addTask
>
> 任务报文：

```json
{
    "modelProcessCode": "XJBY",//下架搬运XJBY；上架搬运 SJBY；货架回库 HJHK
    "priority": "6",//优先级
    "fromSystem": "WMS",
    "orderId": "13706888",
    "taskOrderDetail": [
        {
            "shelfNumber": "A20001",
            "taskPath": "12345678"  //(工作台点位)
        }
    ]
}
```

### 继续任务接口（货架不放下）

```json
{
"orderId":"",
"shelfNumber":"B0622"//任务单号和货架编号任意选择其一即可
}
```

设备到达工作台后上架，上架完成后调用继续任务，货架回库



## 回库任务（货架已放下）

> 请求地址：http://10.68.2.19:7000/ics/taskOrder/addTask
>
> 任务报文：

```json
{
    "modelProcessCode": "HJHK",  //业务流程模板编号
    "priority": 6,   //优先级  可不传
    "fromSystem": "wms",  //任务来源系统
    "orderId": "9667666",  //任务号
    "taskOrderDetail": [
        {
            "taskPath": "",    //(不用发)
            "shelfNumber": "A0887",  //货架编号
    		"deviceNum":""    //设备编号（预占情况下需要下发设备编号）
        }
    ]
}
```
