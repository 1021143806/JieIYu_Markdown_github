# 个人知识库新目录结构设计

## 设计原则

1. **分类清晰**：按照内容类型和使用场景进行分类
2. **层级合理**：控制在3-4层以内，便于导航
3. **命名规范**：使用数字前缀+中文名称，便于排序和理解
4. **扩展性强**：预留扩展空间，便于后续添加新内容

## 资源文件处理原则

1. **图片文件**：统一移动到 `06-资源素材/01-图片资源/` 下，按来源分类
2. **附件文件**：根据关联内容移动到相应目录的资源子目录
3. **模板文件**：统一放到 `06-资源素材/02-文档模板/`
4. **工具软件**：统一放到 `06-资源素材/03-软件工具/`

## 命名规范

1. **目录命名**：`数字前缀-中文名称`格式
2. **文件命名**：避免特殊字符，使用有意义的名称
3. **资源文件**：保持原始名称，必要时添加前缀标识来源
